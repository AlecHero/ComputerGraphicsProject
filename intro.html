Computer Graphics Project: Bézier Curve Drawing Tool

This project implements an interactive Bézier curve editor using WebGL. Users can create and manipulate curves through control points. This project demonstrates the use of curve mathematics, user interaction, and real-time rendering.

Implemented Features

Core Drawing Features

  • Interactive Bézier curve creation
  • Real-time curve manipulation
  • Control point snapping
  • Adjustable curve thickness

Styling & Effects

  • Custom color selection
  • Shape filling functionality
  • High-quality antialiasing
  • WebGL-based rendering

User Interface

  • Undo/Redo functionality
  • Intuitive mouse controls
  • Keyboard shortcuts
  • Canvas image export
