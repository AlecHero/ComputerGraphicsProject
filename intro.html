Computer Graphics Project: Bézier Curve Drawing Tool

This project implements an interactive Bézier curve editor using WebGL. Users can create and manipulate curves through control points. This project demonstrates the use of curve mathematics, user interaction, and real-time rendering.

Technical Demonstration

Note:

Although creating control points for Bézier curves can be a bit confusing, the idea is fairly simple. The first mousedown creates the starting point of the curve. The coming mouseup creates the first control point on the curve. The next mousedown creates the ending point, and mouseup gives another control point. If you wish to create a quadratic curve, you can either do a full keypress in the beginning or end. A full keypress in both the beginning and end will create a straight line.
